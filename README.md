# aozorabunko-json

[青空文庫](https://www.aozora.gr.jp/) を JSON に変換する

- `index_pages/list_person_all_extended_utf8.csv` をもとに次のデータを収集：
  - 本
  - 著者
  - 本 対 著者

## 実行

1. [青空文庫のリポジトリ](https://github.com/aozorabunko/aozorabunko) を clone する
2. `$ cargo run <青空文庫のリポジトリへのパス>`
